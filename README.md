# 💳 Credit Card Fraud Detection System

🚀 A Machine Learning project that detects fraudulent transactions using real-world banking data.

---

## 🔥 Key Highlights

* Detects fraud using ML models
* Handles imbalanced data using SMOTE
* Interactive Streamlit dashboard
* Industry-style ML pipeline

---

## 📊 Problem

Credit card fraud leads to huge financial losses. Detecting fraud is challenging due to **highly imbalanced data**.

---

## 💡 Solution

Built a fraud detection system using:

* Data preprocessing
* SMOTE (imbalance handling)
* Logistic Regression
* Evaluation using precision & recall

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

---

📊 Dataset
Source: Kaggle (ULB Credit Card Fraud Dataset),
Total Transactions: 284,807,
Fraud Cases: 492,
Highly imbalanced dataset.
